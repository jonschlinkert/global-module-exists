> Returns true if a npm package exists in global node_modules.

## Install

Install with [npm](https://www.npmjs.com/):

```sh
$ npm install --save global-module-exists
```

## Usage

```js
var exists = require('global-module-exists');
console.log(exists('generate'));
//=> true
```
